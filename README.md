# pms-2025-spring
ホテル管理システム  - Python コンソールアプリケーション Or GUIアプリケーション

## プロジェクト名
ホテルの予約・空き部屋確認ソフト

## 開発環境
- Python 3.x
- GUI Framework: tkinter
- テストフレームワーク: 目視確認
- プラットフォーム: PC
- 開発環境: VScode

## フォルダ構造
- source/: ソースコード
- 設計/: 設計文書

## 概要
お客様の情報（氏名、メール、人数、部屋数、部屋種、宴会情報）を入力する。  
そこから該当する部屋が空いてるかどうか探索を行う。  
利用可能であれば部屋を予約済みに変更。  
その後確認のメールの発送とjsonに顧客情報の出力をする。  
利用不可の場合は空いてる日付などを表示する。
